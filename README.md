# SAE15 — Analyse du RGA

Projet réalisé dans le cadre de la SAE15 du BUT Réseaux & Télécommunications.

## Objectif
Exploiter les données du Référentiel Général des Armes (RGA) issu de data.gouv afin
d’analyser les caractéristiques des armes, les fabricants et la provenance.

## Technologies utilisées
- Python (lecture et structuration CSV)
- HTML / CSS / JavaScript
- Leaflet (carte interactive)
- Chart.js (graphiques)
